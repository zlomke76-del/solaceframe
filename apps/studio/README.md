# SolaceFrame Studio V2

This is the first real deployable SolaceFrame workspace shell.

## V2 Delta

- Full Next.js App Router structure
- Premium governed media workspace UI
- Continuity, governance, provenance, storyboard, and audit sections
- Mobile responsive layout
- Patched Next.js 14.2.35 dependency

## Vercel

Set Root Directory to:

```txt
apps/studio
```

Build command:

```txt
npm run build
```

Install command:

```txt
npm install
```
