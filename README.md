# SolaceFrame

Governed synthetic media infrastructure for persistent image and video generation.

## Core Principles

- Continuity-aware generation
- Runtime governance
- Persistent character/world memory
- Provenance + audit lineage
- Long-form coherence
- Identity-bound generation
- Enterprise-safe synthetic media

## Architecture

apps/
- studio -> Next.js workspace
- api -> orchestration API
- admin -> governance console

packages/
- governance
- continuity
- provenance
- prompts
- models
- safety

workers/
- image-worker
- video-worker
- continuity-worker
